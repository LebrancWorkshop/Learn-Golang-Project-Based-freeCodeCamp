- Details:

      - DEF (Defense)
      - INT (Intelligence)

      - HP (Health Point)
      - SP (Skill Point)
      - ATK (Attack)
      - DEF (Defense)
      - INT (Intelligence)
    - Method:
      - attack()

    - Attribute:
      - HP (Health Point)
      - MP (Mana Point)
      - ATK (Attack)
      - DEF (Defense)
      - INT (Intelligence)
    - Method:
      - attack()

    - Attribute:
      - HP (Health Point)
      - SP (Skill Point)
      - ATK (Attack)
      - DEF (Defense)
      - INT (Intelligence)
    - Method:
